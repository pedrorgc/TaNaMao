# Estrutura do Frontend (React)

- **components/ui/** → Componentes reutilizáveis (botões, inputs, modais)
- **components/layout/** → Navbar, Footer, Sidebar, etc.
- **pages/** → Páginas da aplicação (Home, Login, Dashboard)
- **hooks/** → Hooks customizados
- **context/** → Context API (ex.: AuthContext)
- **services/** → Integração com API (axios, endpoints)
- **utils/** → Funções auxiliares
- **styles/** → Estilos globais
